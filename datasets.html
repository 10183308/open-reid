reid.datasets¶

reid.datasets.get_dataset(name, root, *args, **kwargs)[source]¶

Create a dataset instance.

Parameters:
  • name (str) – The dataset name. Can be one of ‘viper’, ‘cuhk01’, ‘cuhk03’, ‘market1501’, and ‘duke’.
  • root (str) – The path to the dataset directory.
  • split_id (int, optional) – The index of data split. Default: 0
  • num_val (int or float, optional) – When int, it means the number of validation identities. When float, it means the proportion of validation to all the trainval. Default: 100 for ‘cuhk01’ and ‘cuhk03’, 0.3 for the others
  • download (bool, optional) – If True, will download the dataset. Default: False
